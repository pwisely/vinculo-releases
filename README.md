# Vínculo Releases

This is the official public Windows release channel for **Vínculo Genealogy Intelligence**, published by **Wisely Enterprises LLC**.

This repository intentionally contains only:

- publicly downloadable, signed Vínculo installers;
- installer blockmaps used for safe in-place updates;
- `latest.yml` update metadata; and
- public release notes and installer checksums.

It must never contain application source code, private genealogy records, customer vaults, GEDCOM files, DNA data, credentials, signing secrets, API keys, or unpublished research.

## Release status

The update channel is reserved, but no production installer has been published yet. A release is published only after its installer and installed application have matching, valid, timestamped Windows signatures from Wisely Enterprises LLC and the release metadata has been verified against the exact installer bytes.

Vínculo customer genealogy data remains on the customer’s computer and is not stored in this repository.
